5.2 Flow by Phase



Phase

Flow

Purpose

Ingestion

AX → Gateway → Synapse → ADLS

Extract source data into lake

Transformation

ADLS → Databricks → dbt → Delta Gold

Clean and model data

Consumption

Gold → SQL → AS → Power BI

Deliver trusted reporting



Phase 1 – Ingestion: Extract source data from AX and land it into the data lake.



Phase 2 – Transformation: Clean, transform, and model raw data into business-ready datasets.



Phase 3 – Consumption: Deliver trusted semantic models and dashboards to business users.
